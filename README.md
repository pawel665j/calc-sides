# calc-sides
Для него нужен только Python 3.9 и выше
Простой и удобный калькулятор соотношений сторон с визуализацией и справочником стандартных разрешений. Автоматически рассчитывает недостающую сторону, округляет до кратных 8 и показывает актуальные форматы (16:9, 9:16, 4:3 и др.).

A simple and user-friendly aspect ratio calculator with live preview and a reference list of standard resolutions. Automatically computes the missing dimension, rounds to multiples of 8, and displays common formats (16:9, 9:16, 4:3, etc.).

Yes, you can use it at home, no need to search for this calculator on the Internet, it takes up little space and is always at hand. I thought about creating this calculator when the site where I did such calculations disappeared, searching for a new site, etc., it turned out to be faster to make a small script,
